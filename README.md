# learning-scala3

A project-driven learning lab for Scala 3. Instead of reading through books end to end, the plan here is to combine a knowledge map with a sequence of progressively harder small projects — clear Scala first, then functional Scala, and only then highly abstract Scala.

## Navigation

- [roadmap.md](roadmap.md) — the knowledge map. What to learn, organized by stage.
- [projects/README.md](projects/README.md) — the action plan. Sixteen projects (with a Cats Effect / ZIO branch in the middle) from CLI tool to tagless final app, with recommended order and iteration method.

Each project has its own folder under `projects/` with a self-contained README.
